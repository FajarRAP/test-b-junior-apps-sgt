## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/FajarRAP/test-b-junior-apps-sgt.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd test-b-junior-apps-sgt
   ```
3. **Install dependencies:**
   ```bash
   flutter pub get
   ```
4. **Run the app:**
   ```bash
   flutter run
   ```

## Requirements

- Flutter SDK
- Dart SDK
